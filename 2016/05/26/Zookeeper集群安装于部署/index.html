Zookeeper集群安装于部署

发表于 2016-05-26   |   分类于 Hadoop   |  

Zookeeper我就不多做介绍了，具体请看
维基百科，或者官网介绍

1、准备环境

安装jdk，安装jdk，安装jdk

　　重要的事情说三遍，因为我就是没有装，导致弄了好久老是报错
使用zkServer.sh status 看出角色时报Error contacting service. It is probably not running.的错误

　　因为Zookeeper是java写的，所以需要java的环境。请到oracle官网下载最新版的jdk，并且解压安装，解压的话最好放到/usr/lib/jdk/下
，接着到Apache官网下载 Zookeeper。
　　然后，修改hosts文件，把主机跟ip地址映射起来，

如果嫌麻烦的话，可以不用这一步，待会配置是直接填ip就行

2、解压安装

tar -zxvf zookeeper-3.4.6.tar.gz

3、复制配置文件

cp /root/zookeeper-3.4.6/conf/zoo_sample.cfg /root/zookeeper-3.4.6/conf/zoo.cfg

4、修改配置文件zoo.cfg

#建立data文件夹，用于存放数据的
mkdir /root/zookeeper-3.4.6/data

vim /root/zookeeper-3.4.6/conf/zoo.cfg

#把下面对应的值改了就行
dataDir=/root/zookeeper-3.4.6/data

server.1=itr-mastertest01:2888:3888
server.2=itr-mastertest02:2888:3888
server.3=itr-nodetest01:2888:3888

5、进入data目录

cd /root/zookeeper-3.4.6/data
vim myid
1

#1,代表第server.1台server
!wq

#保存退出

6、复制到其他集群服务器上

scp -rq /root/zookeeper-3.4.6 itr-mastertest02:/root/
scp -rq /root/zookeeper-3.4.6 nodetest01:/root/

7、修改myid的值

[root@mastertest02 data]# vim myid

#写入2
[root@server3 data]# vi myid

#写入3

8、启动zookeeper

[root@itr-mastertest01 ~]# zk zookeeper-3.4.6/bin/zkServer.sh start
[root@itr-mastertest02 ~]# zk zookeeper-3.4.6/bin/zkServer.sh start
[root@itr-nodetest01 ~]# zk zookeeper-3.4.6/bin/zkServer.sh start

9、查看状态

[root@itr-mastertest01 ~]# zk zookeeper-3.4.6/bin/zkServer.sh status
