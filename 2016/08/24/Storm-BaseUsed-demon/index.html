建立Spout

Tuple的概念。
Storm中，基本元数据是靠Tuple才承载的。或者说，Tuple是数据的一个大抽象。它要求实现类必须能序列化。
Tuple中以List存放Values，List的Index按照new Values(obj1, obj2,…)的参数的index,例如我们emit(new Values(“v1”, “v2”)), 那么Tuple的属性即为：{ [ “v1” ], [ “V2” ] }

集群运行

用 mvn clean install 命令编译，然后把target目录下生成的 storm-samples-jar-with-dependencies.jar 拷到nimbus机器上，执行

cd apache-storm-1.0.2/bin

./storm jar /home/hadoop/storm-samples-jar-with-dependencies.jar top.guinguo.storm.ExclaimBasicTopo test

然后可以到web UI里面查看详情信息
点进去test，可以看到

看到spout已经发射了76780个tuple了，而id为exclaim的bolt已经接收了72780个tuple了，
而print没有输出，emit为0。
